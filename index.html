millis()関数

サンプルコードの結果

サンプルコード

　　 void draw(){

　　 　int timer=millis();

　　 　println(timer);

　　 }

説明文

　millis()関数はプログラムがスタートしてからの経過時間を値として返す関数。

　

　processingはプログラムがスタートしてから経過時間をカウントしている。

　その数値を値として出力するのがmillis()関数である。

　

　単位はミリ秒（1000分の1秒）で、1秒が経過すると1000、5秒なら5000、1分で60000となる。

　結果では、9秒から10秒が経った頃である。
